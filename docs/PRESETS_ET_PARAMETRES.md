# Presets et paramètres

VirtualAudioMix peut mémoriser vos configurations audio.

## Presets

Un preset contient:

- les blocs;
- les liens;
- les gains;
- les positions des blocs;
- l'état activé/désactivé des blocs;
- les canaux utilisés.

## Créer un preset

1. Préparez votre graphe.
2. Ouvrez la section `Presets`.
3. Créez ou sauvegardez le preset.

Le preset pourra être rappelé plus tard.

## Charger un preset

Sélectionnez un preset dans le menu `Presets`.

Le graphe est restauré automatiquement.

Si un périphérique n'est plus présent, le bloc peut apparaître comme absent. Rebranchez le périphérique ou choisissez une autre source/sortie.

## Import/export JSON

VirtualAudioMix permet d'exporter des presets en JSON.

Utilisations possibles:

- sauvegarder une configuration;
- transférer un preset vers un autre PC;
- partager une configuration.

Les exports peuvent être faits pour:

- le preset courant;
- tous les presets.

## Dossier d'export

Le dossier d'export peut être choisi dans les paramètres.

Si aucun dossier n'est choisi, VirtualAudioMix utilise le dossier `Téléchargements`.

## Démarrage avec Windows

Dans les paramètres, vous pouvez activer ou désactiver:

- le lancement de VirtualAudioMix avec Windows;
- le démarrage automatique du moteur audio après chargement du driver BAD.

## Zone de notification

Quand VirtualAudioMix est réduit, il peut rester dans la zone de notification Windows.

Actions disponibles:

- clic gauche: restaurer la fenêtre;
- clic droit: ouvrir le menu contextuel;
- changer de preset;
- démarrer ou arrêter le moteur audio;
- quitter réellement l'application.
